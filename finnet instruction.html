FINLEARN: Member Portal User Guide

How to login

1. Go to the portal website

2. Enter your member ID

3. Enter your password

4. Click "Login"

How to view member statement

1. Navigate to "My Account"

2. Click "View Statement"

3. Select statement period

How to claim my benefits

1. Go to "Benefits" section

2. Select benefit type

3. Submit claim form

How to view my profile

1. Click "My Profile"

2. View personal details

How to edit my profile

1. Go to "My Profile"

2. Click "Edit Profile"

3. Update information

4. Save changes

How to view transaction history

1. Navigate to "Account Activity"

2. Select date range

3. View transaction details

How to view scheme details

1. Go to "Plan Information"

2. Select scheme type

3. View terms and conditions

How to vote

1. Access "Member Voting"

2. Review options

3. Submit vote

How to change my password

1. Go to "Account Settings"

2. Click "Change Password"

3. Enter new password

4. Confirm changes

How to view news and events

1. Click "News & Updates"

2. Browse latest announcements

How to logout

1. Click "Account Options"

2. Select "Logout"
